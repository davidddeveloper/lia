LinkedIn AI Assistant - Installation Guide

This guide will walk you through installing and setting up the LinkedIn AI Assistant Chrome extension.

1 Download the Extension

First, download the extension files from the GitHub repository or the provided ZIP file.

If you received a ZIP file, extract it to a folder on your computer.

2 Open Chrome Extensions Page

Open Google Chrome and navigate to the Extensions page using one of these methods:

  • Type chrome://extensions in the address bar and press Enter
  • Click the three dots menu in the top-right corner, select "More tools", then "Extensions"

3 Enable Developer Mode

In the top-right corner of the Extensions page, toggle on "Developer mode".

4 Load the Extension

Click the "Load unpacked" button that appears after enabling Developer mode.

Navigate to the folder where you extracted the extension files and select that folder.

5 Verify Installation

The LinkedIn AI Assistant should now appear in your list of extensions.

You should see the extension icon in your browser toolbar (top-right corner).

Setting Up the Extension

1 Get an OpenAI API Key

The extension requires an OpenAI API key to function. If you don't have one:

  1. Go to platform.openai.com/signup
  2. Create an account or sign in
  3. Navigate to the API keys section
  4. Create a new API key
Note: Keep your API key secure and never share it publicly. Using the OpenAI API will incur charges based on your usage.

2 Configure the Extension

Click the LinkedIn AI Assistant icon in your browser toolbar to open the settings popup.

  1. Enter your OpenAI API key in the designated field
  2. Select your preferred writing tone
  3. Choose your industry
  4. Toggle which features you want to enable
  5. Click "Save Settings"

Using the Extension

1 Creating Posts

When creating a new LinkedIn post, you'll see an "AI Assist" button near the post editor.

Click this button to generate post suggestions based on your settings.

2 Replying to Comments

When replying to a comment, you'll see an "AI Reply" button near the comment box.

Click this button to generate thoughtful reply suggestions.

3 Improving Your Posts

On your own published posts, you'll see an "Improve Post" button.

Click this button to get AI-powered suggestions to enhance your post's engagement and effectiveness.

Privacy Note: Your content is sent to OpenAI's API for processing. Please review OpenAI's privacy policy for more information on how they handle data.

Troubleshooting

Extension Not Working

  • Make sure you've entered a valid OpenAI API key
  • Check that you have sufficient credits in your OpenAI account
  • Try refreshing the LinkedIn page
  • Ensure the extension is enabled in Chrome

Buttons Not Appearing

  • LinkedIn may have updated their UI. Check for extension updates
  • Try refreshing the page
  • Disable and re-enable the extension
